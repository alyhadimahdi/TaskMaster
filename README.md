# TaskMaster

A modern productivity web application built for **CSCI390 — Web Programming**.

## Tech Stack

- ReactJS
- Tailwind CSS
- React Router
- Vite

## Pages

- **Home** — App overview and features
- **About** — Project purpose and technologies
- **Services** — Feature showcase with interactive task manager
- **Contact** — Contact form

## Setup

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
cd frontend
npm run build
```

## Deployment

Build the project, then deploy the `frontend/dist` folder to GitHub Pages, Vercel, or Netlify.
